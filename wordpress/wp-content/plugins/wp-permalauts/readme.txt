Deutsche Umlaute in Permalinks von WordPress

== Description ==

**Plugin home: [WP Permalauts](http://permalauts.de/)**

In WordPress werden die Permalinks ganz simpel auf den erlaubten ASCII-Zeichensatz heruntergebrochen:
ä = a, ö = o, ü = u und ß = s.

Mit installiertem und aktivem Permalauts folgt:
ä = ae, ö = oe, ü = ue und ß = ss.

[Permalauts: Beschreibung](http://permalauts.de/)
